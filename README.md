# PYCOM --- Python DICOM Processing Toolkit for BEAMnrc/DOSXYZnrc.

PYCOM is a collection of DICOM processing tools coded in the Python programming 
language for assisting MC simulation using BEAMnrc/DOSXYZnrc using RT plans 
generated from a TPS system. It currently has tools: 

(a) pycom.py: generating input files automatically for BEAMnrc/DOSXYZnrc 
from DICOM RT Plans (DICOM RP) exported from a Treatment Planning System (TPS). 

(b) pycom_dose.py: converting the dose distribution generated by DOSXYZnrc 
in 3ddose format to DICOM RT Dose (DICOM RD) format so that it can be imported 
by a DICOM processing software.

(c) pycom_anon.py: anonymizing DICOM files for patient privacy. 

See the comments in the files for the usage of the tools.

Note: The current version is for BEAMnrc version 2011. 
Tools for BEAMnrc version 2013 is in development.


This version was developed under Python2. Here are the steps to setup Python2 environment for runningthis package:

1. Install a latest Linux OS with Python2.7 included, such as Ubuntu 22.04 or RHEL7 or RHEL8. We use Ubuntu 22.04 here.

2. Set up GIT

```
sudo apt install git
sudo apt install gh
gh auth login
gh repo clone https://github.com/lixinzhan/pycom.git
```

2. Run commands below to set up a Python2.7 virtual environment

```
sudo apt install python2
sudo apt install python2-pip
sudo pip2 install virtualenv
virtualenv pycom/venv2
```

3. Set up the PYCOM running environment

```
cd pycom
source rc.py2env
pip2 install -r requirements.txt
```
